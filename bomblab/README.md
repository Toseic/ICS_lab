# bomb自动本地化


`safebomb.py`可以将bomb自动本地化
使用方式：
将`safebomb.py`与`bomb`放在同一个文件下，运行

```bash
$ python3 ./safebomb.py
```
即可

生成的`safebomb`如果运行显示权限不够，可以运行：

```bash
$ sudo chomd 775 ./safebomb
```
